GeoEditor Controls

  • Draw: Polygon, Line, Rectangle, Circle, Marker, Ellipse, Freehand
  • Edit: Select, Drag, Edit vertices, Rotate, Cut, Delete
  • Advanced: Scale, Copy, Split, Union, Difference, Simplify, Lasso
  • File: Open GeoJSON, Save GeoJSON

Selection

  • Select mode: Click features to select them
  • Shift+Click: Add/remove from selection
  • Union: Click tool, select polygons, press Enter
  • Difference: Click tool, select base + subtract polygons, press Enter

Keyboard Shortcuts

  • Ctrl+C: Copy selected
  • Ctrl+V: Paste
  • Delete: Delete selected
  • Enter: Execute union/difference
  • Escape: Cancel / Clear selection
