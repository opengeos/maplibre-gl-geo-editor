CDN Usage Example

This example loads all dependencies via CDN. No build step required.

Draw Tools

  • Polygon, Line, Rectangle
  • Circle, Marker, Freehand
  • Ellipse, Circle Marker

Edit Tools

  • Select, Drag, Edit, Rotate
  • Cut, Delete, Scale, Copy
  • Split, Union, Difference
  • Simplify, Lasso

File Tools

  • Open GeoJSON file
  • Save as GeoJSON

Keyboard Shortcuts

  • Ctrl+C Copy selected
  • Ctrl+V Paste
  • Delete Delete selected
  • Enter Execute operation
  • Escape Cancel / Clear
